# Extension Methods

A lot of functionality of the Data Structures package comes from extension methods that provide enhanced support for common container types and interfaces. Extension methods are available for the following:

- [Array](xref:Zigurous.DataStructures.ArrayExtensions)
- [Comparable](xref:Zigurous.DataStructures.ComparableExtensions)
- [Dictionary](xref:Zigurous.DataStructures.DictionaryExtensions)
- [HashSet](xref:Zigurous.DataStructures.HashSetExtensions)
- [List](xref:Zigurous.DataStructures.ListExtensions)
- [SortedSet](xref:Zigurous.DataStructures.SortedSetExtensions)
- [String](xref:Zigurous.DataStructures.StringExtensions)
